GitHub Pages — tek domain (köprü)

1) Depoda bu klasörü site kökü yapın:
   - Repo Settings → Pages → Build: Deploy from branch
   - Klasör: /marka-portal (veya tüm siteyi yalnızca bu klasörün içeriği olacak şekilde ayrı repo)

2) 404.html, index.html ile aynı kopyadır; GitHub bilinmeyen yolları 404.html ile sunar, böylece köprü açılır.

3) GitHub Pages yalnızca statik dosya sunar. /api/v1/... burada çalışmaz; API için Render vb. ayrı servis gerekir.
   kocaeli.html bu ortamda boş/uyarı modunda açılır. Market kartını tam üretimde harici URL yapmak için
   index.html içindeki PORTAL_BRANDS satırını düzenleyin.
